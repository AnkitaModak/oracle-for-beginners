# Oracle for Beginners

A beginner-friendly open-source Oracle Database tutorial repository with SQL, PL/SQL, exercises, and real-world examples.

---

## 📚 What You Will Learn

* Oracle Database basics
* SQL fundamentals
* Table creation
* Constraints
* Joins
* Functions
* PL/SQL
* Procedures
* Triggers
* Transactions
* Real-world practice examples

---

## 🚀 Who Is This For?

This repository is designed for:

* Beginners learning Oracle Database
* Students
* Developers switching to Oracle
* Anyone wanting practical Oracle tutorials

---

## 📂 Repository Structure

```bash
docs/          -> Tutorials and explanations
examples/      -> SQL examples
exercises/     -> Practice problems
projects/      -> Mini real-world projects
```
