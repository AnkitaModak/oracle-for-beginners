# Contributing Guide

Thank you for contributing to Oracle for Beginners.

## Ways You Can Contribute

* Improve tutorials
* Fix typos or mistakes
* Add SQL examples
* Add exercises
* Improve documentation

---

## Contribution Steps

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Commit changes
5. Open a Pull Request

---

## Guidelines

* Keep explanations beginner friendly
* Add comments to SQL code
* Use clear formatting
* Test SQL queries before submitting

---

## Beginners Welcome

Even small contributions are valuable.
