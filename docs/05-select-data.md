# Selecting Data in Oracle

The `SELECT` statement is used to retrieve data from a table.

---

## Basic Syntax

```sql id="xv7r7z"
SELECT column_name
FROM table_name;
```

---

## Select All Columns

```sql id="wb8m5f"
SELECT * FROM students;
```

`*` means all columns.
